# Letterboxd Stream Finder

A web app that helps Letterboxd users find which movies from their watchlist are available on their streaming services.

## Planned Features

- Search by Letterboxd username
- Import user watchlist
- Select streaming services
- Filter by streaming availability
- Sort by rating and watchlist age
- View movie posters and metadata

## Tech Stack

- React
- Vite
- Node.js
- Express
- TMDB API
